## 정렬하기

> `dplyr` 패키지에 있는 `arrange`를 이용해 변수를 추출하는 방법에 대해 알아본다.



### data

```R
exam <- read.csv('csv_exam.csv')
head(exam)
#   id class math english science
# 1  1     1   50      98      50
# 2  2     1   60      97      60
# 3  3     1   45      86      78
# 4  4     1   30      98      58
# 5  5     2   25      80      65
# 6  6     2   50      89      98
```



### 정렬 예제

* 수학을 기준으로 오름차순 정렬

  ```R
  exam %>% arrange(math) %>% head(10)
  ```

  ```R
  # 결과

  ```

* 수학을 기준으로 내림차순 정렬

  ```R
  exam %>% arrange(desc(math)) %>% head(10)
  ```

  ```R
  # 결과
  ```

* 반(오름차순), 수학(내림차순)으로 정렬

  ```R
  exam %>% arrange(class, desc(math)) %>% head(10)
  ```
